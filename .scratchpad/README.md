# .scratchpad

Working directory for throwaway, local-only content related to this repo: screenshots, progress notes, UX review drafts, personal notes, TODO lists, or anything else you want to jot down while working here.

Nothing in here is part of the app or its documentation. Everything except this file is gitignored, so contents stay local and won't show up in `git status` or get committed — put whatever's useful to you in here without worrying about it leaking into a commit.
